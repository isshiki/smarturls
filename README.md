# SmartURLs – Copy and Open

Copy all URLs (with titles) as Markdown, HTML, TSV, JSON, or custom text.  
It also works the other way around — opening URLs from text or the clipboard, where Smart Parser mode automatically detects any format.

---

## ✨ Features

- Copy tab addresses (URLs) with optional titles
- Output formats: Markdown, HTML, TSV, JSON, or custom template
- Open URLs from clipboard or text area
- Smart Parser mode (enabled by default) automatically detects and handles various formats
- Options for duplicate removal, HTTP/HTTPS filtering, and pinned-tab exclusion
- Minimal permissions, Manifest V3 compliant

---

## Usage

1. Click the extension icon to open the popup.  
2. Press **Copy URLs** to copy all tab URLs in the selected format.  
3. Press **Open URLs** to open URLs detected from the clipboard or entered text.

---

## Development

This project is a Chrome extension built with Manifest V3.  
To load it locally:

1. Clone this repository.  
2. Go to `chrome://extensions` in Chrome.  
3. Enable **Developer mode**.  
4. Click **Load unpacked** and select the `smarturls` folder.
